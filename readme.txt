== Description ==

Price calculator to your website.
Плагин позволяет создать калькулятор цен для того, чтобы Ваши пользователи смогли посчитать стоимость Ваших услуг или товаров.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Price calculator to your website'
3. Activate the plugin from your Plugins page.
4. ...

= From WordPress.org =

1. Download 'Price calculator to your website'.
2. Upload the 'Price calculator to your website' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Price calculator to your website' from your Plugins page.
4. ...
