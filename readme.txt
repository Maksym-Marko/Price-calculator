Калькулятор цен для Вашего сайта.
Плагин позволяет создать калькулятор цен для того, чтобы Ваши пользователи смогли посчитать стоимость Ваших услуг или товаров.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Калькулятор цен на сайт'
3. Activate the plugin from your Plugins page.
4. ...

= From WordPress.org =

1. Download 'Калькулятор цен на сайт'.
2. Upload the 'Калькулятор цен на сайт' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Калькулятор цен на сайт' from your Plugins page.
4. ...
